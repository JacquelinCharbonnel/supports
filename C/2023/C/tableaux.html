Tableaux

1. Tableau à 1 dimension

T t[N] définit un tableau t de N éléments de type T, indicés de 0 à N-1.

E/S
#define N 10

int i ;
int tab[N] ;

/* initialisation du tableau à partir de valeurs lues au clavier */
for (i=0 ; i<N ; i++) scanf("%d",&tab[i]) ;

/* sortie du tableau */
for (i=0 ; i<N ; i++) printf("%d\n",tab[i]) ;
Tri (bubble sort)
#define VRAI 1
#define FAUX 0

void tri(int t[])
{
  int i,permut, x ;

  do {
       permut = FAUX ;

      for (i=0 ; i<N-1 ; i++)
      {
        /* on permute ici les 2 éléments qui ne sont pas dans le bon ordre */
      }
  } while (permut) ;
}

  • On ne peut pas recopier un tableau avec l’opérateur = : cela produit une erreur de compilation. Pour cela, on peut utiliser la fonction memcpy.

  • On ne peut pas comparer 2 tableaux avec l’opérateur == : cela ne produit pas d’erreur, mais ne donne pas le résultat escompté (en fait, cela teste si les 2 tableaux commencent à la même adresse, ce qui généralement est faux). Pour cela, on peut utiliser la fonction memcmp.

2. Tableau à 2 dimensions

int mat[3][3] = { { 1,0,0 }, {0,1,0}, {0,0,1} } ;

for (i=0 ; i<3 ; i++)
{
  for (j=0 ; j<3 ; j++) printf("%3d ",mat[i][j]) ;
  printf("\n") ;
}

Ce programme sort :

  1   0   0
  0   1   0
  0   0   1
