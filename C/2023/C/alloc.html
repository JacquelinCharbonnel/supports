Allocation de mémoire

Gérer dynamiquement la mémoire se fait via des fonctions de la bibliothèque standard :

  • d’allocation : malloc

  • de libération : free

Exemple
#include <stdio.h>
#include <stdlib.h>

void tri(float []) ;

int main()
{

  float *tab ;

  tab = malloc(n*sizeof(float)) ;
  if (tab==NULL) { perror("malloc") ; exit(1) ; }

  for (i=0 ; i<n ; i++)
