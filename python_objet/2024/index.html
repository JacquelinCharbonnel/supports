Programmation Objet en Python

Ce support de cours s’adresse aux étudiants en informatique.

La programmation orientée objet (POO) est un paradigme de programmation, c’est-à-dire une façon de formaliser la solution logicielle d’un problème et d’organiser le code informatique.

Ce support explique comme programmer en Python en suivant les prescriptions de la POO.
