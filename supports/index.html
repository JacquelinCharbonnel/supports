Archive de supports

Linux dans son contexte

Support destiné au grand public. Expose les origines de Linux, ses composantes, ses caractéristiques, comment il est développé et packagé, et par qui.

Langage C

Support de cours pour les licences informatiques. Apprentissage à la programmation en C, avec, en seconde partie, quelques cas d’usage de programmation système sous Unix. Dernière mise à jour en 2023.

Programmation objet en python

Support de cours pour les masters Data Science. Guide les étudiants connaissant la programmation en python à adopter une approche objet de la conception d’une application, et de son développement en python. Dernière mise à jour en 2024.

Programmation concurrente

Support de cours pour les masters Data Science. Explique ce qu’est la programmation concurrente et son intérêt. Clarifie les notions de parallèlisme, asynchronisme, multi-tâche, coroutine. Application pratique en python. Dernière mise à jour en 2024.

XML & JSON

Supports de cours pour les masters Data Science. Un aperçu des formats XML et JSON. Dernière mise à jour en 2023.

Usage d’Antora

Antora est un générateur de site web statique. Je l’ai utilisé pour mes supports de cours, je n’ai pas trouvé mieux. Ce support, qui date de 2023, explique comment je m’en suis servi.
